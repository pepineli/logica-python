##  Sobre o Projeto

Este repositório contém **10 exercícios práticos** que desenvolvi para praticar os fundamentos da lógica de programação usando Python.

---

##  Exercícios

| # | Arquivo | Descrição |
|---|---------|-----------|
| 1 | `hello_word.py` | Programa inicial de boas-vindas |
| 2 | `tabuada.py` | Gera a tabuada de um número |
| 3 | `par_impar.py` | Verifica se um número é par ou ímpar |
| 4 | `primo.py` | Verifica se um número é primo |
| 5 | `fatorial.py` | Calcula o fatorial de um número |
| 6 | `soma_numeros.py` | Soma de 1 até N |
| 7 | `maior_numero.py` | Encontra o maior entre 3 números |
| 8 | `contagem_regressiva.py` | Contagem regressiva de 10 a 1 |
| 9 | `media_notas.py` | Calcula média de 4 notas com conceito |
| 10 | `palindromo.py` | Verifica se uma palavra é palíndromo |
